# PRRX Dashboard

A complete dashboard with Google Login, Live Chat, and Feature Management.

## Features
- Google Authentication
- Real-time Dashboard Analytics
- Live Chat System
- Feature Management Panel
- Responsive Design

## Setup
1. Clone this repository
2. Upload to GitHub Pages
3. Enable Google Sign-in in Firebase Console
4. Add your domain to authorized domains

## Firebase Setup
1. Create a Firebase project
2. Enable Google Authentication
3. Add your domain to authorized domains
4. Update firebase-config.js with your credentials

## Deployment
Simply push to GitHub and enable GitHub Pages in repository settings.

## Technologies
- HTML5/CSS3
- JavaScript (ES6+)
- Firebase Authentication
- Firebase Realtime Database
- Chart.js
