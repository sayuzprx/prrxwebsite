Settings Menu

Complete Configuration

Customize PRRX to your preference with the comprehensive settings menu. Optimize performance and appearance.

✨ Settings Options:

  • STREAMER MODE - Hide sensitive info for streaming
  • MOUSE BUG FIXED - Fix mouse-related issues
  • HIDE TASK - Hide PRRX from taskbar
  • FIXED ERRORS - Auto-fix common errors
  • AIMBOTS MUTE SOUND - Mute aimbot sound effects
  • CHECK USAGE - Monitor resource usage
  • PRRX LAGGING FIXED - Performance optimization
  • PANEL THEMECOLOR - Change UI theme colors
  • CLEAN TEMP - Clear temporary files
  • EXTRA BOOST / COMBAND BOOST - Performance boosts
  • BIN CLEAN - Clean binary cache

How to Configure:

  1. Open Settings Menu from Main Menu
  2. Enable Streamer Mode if streaming
  3. Select PANEL THEMECOLOR for appearance
  4. Use BOOST options for performance
  5. Click SAVE to keep settings permanently
